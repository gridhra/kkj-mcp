# 入札方式（Bidding Methods）

## Brief

### 主要な入札方式

1. **一般競争入札** - 参加資格を満たす全事業者が参加可能。最も透明性が高い
2. **指名競争入札** - 発注者が指名した事業者のみが参加。事前の名簿登録が必須
3. **随意契約（オープンカウンタ）** - 少額案件向け。見積合わせで即決

### 重要ポイント

- **WTO案件**: 約6.8億円以上の大型案件。公告期間40日以上、広範囲の競争
- **申請締切**: 入札日ではなく「参加資格確認申請締切」が最重要
- **緊急案件**: 公告から締切まで2-3日のケースあり

## Detailed

### 2.1.1 一般競争入札（Ippan Kyoso Nyusatsu）

**概要**
最も標準的かつ透明性の高い方式。一定の条件を満たす全ての事業者が参加可能です。

**メカニズム**
発注機関が公告を出し、参加資格（ランク、地域、実績）を設定します。

**AIへの重要示唆**
この方式における最大のリスクは「参加資格確認申請」の締切です。入札日（金額を入れる日）が1ヶ月先であっても、参加意思を表明する申請締切は公告から10〜15日後であることが大半です。**AIは「入札日」ではなく「申請締切日」をクリティカルパスとしてユーザーに警告しなければなりません。**

**WTO対象案件**
調達額が基準額（SDR換算で約6億8,000万円以上の工事等）を超える場合、「WTO政府調達協定」が適用されます。

- 公告期間が40日以上に延長
- 官報への掲載が義務付けられる
- 広範な地域からの参入（激しい競争）が予想される

AIは「WTO」のタグを検知した場合、長期戦のプロジェクトであることを示唆する必要があります。

### 2.1.2 指名競争入札（Shimei Kyoso Nyusatsu）

**概要**
発注者が特定の業者を選定（指名）して競争させる方式です。

**公募型指名競争入札**
従来は完全なクローズドでしたが、現在は「公募型」が主流です。これは「手を挙げた業者の中から指名する」方式であり、実質的には一般競争入札に近い手続きを踏みますが、技術資料の審査がより厳格になる傾向があります。

**AIへの重要示唆**
検索結果に「指名」の文字がある場合、AIはユーザーに対し「あなたが既に指名願（参加資格審査申請）を提出し、名簿に登載されているか」を確認するよう促すべきです。**名簿外の業者は、どんなに技術力があっても土俵に上がれません。**

### 2.1.3 随意契約（Zuii Keiyaku / Open Counter）

**概要**
競争入札を行わず、特定の業者と契約する方式ですが、少額案件では「オープンカウンタ（見積合わせ）」と呼ばれる形式がとられます。

**メカニズム**
仕様書が公開され、指定期間内に最も安い見積を出した業者が即決します。

**AIへの重要示唆**
**スピードが全てです。** 公告から締切までが2〜3日というケースも珍しくありません。AIはこれを「緊急案件」としてタグ付けし、即時の対応を推奨する必要があります。

## Comprehensive

### 入札方式の法体系と構造的理解

入札公告（Koukoku）を分析する際、AIが最初に識別すべきは「入札方式」です。これがプロジェクトの全工程と競争環境を決定づける「物理法則」となります。

### 一般競争入札の詳細プロセス

**法的根拠**: 地方自治法第234条、地方自治法施行令第167条の6

**標準的なフロー**:
1. 公告（Day 0）- KKJ API等で情報公開
2. 質問期間（Day 1〜10）- 設計図書の閲覧と質疑応答
3. 資格確認申請（Day 14前後）- **【最重要】** 参加表明のデッドライン
4. 入札（Day 20〜25）- 見積金額の提出
5. 開札（Day 25）- 金額の公開と落札候補者の決定
6. 事後審査（Day 26〜30）- 落札候補者の資格要件の詳細審査
7. 契約締結（Day 35）- 正式契約

**WTO政府調達協定の影響**

調達額の基準値:
- 工事: 約6億8,000万円（SDR 4.5 million）
- 物品: 約2,400万円（SDR 130,000）
- 役務: 約2,400万円（SDR 130,000）

WTO案件の特徴:
- 公告期間が40日以上（国内案件の4倍）
- 官報掲載義務
- 国籍による差別禁止（外資系企業も参入可能）
- 英文での質問対応が求められる場合あり

### 指名競争入札の実務

**従来型指名競争入札**
発注者が独自の判断で5〜10社程度を指名。中小規模の自治体や特殊な専門工事で多用されます。

**公募型指名競争入札**
1. 発注者が案件を公告
2. 希望業者が参加申請
3. 発注者が技術資料を審査し、指名業者を選定
4. 指名された業者のみが入札に参加

**名簿登載の重要性**
多くの自治体では、年1回（通常4月または7月）の「入札参加資格審査」で名簿を作成します。この期間を逃すと、その年度の指名型案件には一切参加できません。

AIは検索結果に「指名」が含まれる場合、ユーザーの名簿登載状況を確認するよう促す必要があります。

### 随意契約・オープンカウンタの実態

**法的根拠**: 地方自治法施行令第167条の2

**適用される条件**:
- 少額案件（工事: 250万円以下、物品: 160万円以下など、自治体により異なる）
- 緊急性が極めて高い場合
- 特殊な技術・設備を持つ業者が限定される場合
- 災害復旧など競争入札に付する暇がない場合

**オープンカウンタの流れ**:
1. 公告（通常2〜5日前）
2. 見積書提出期限
3. 即日開封、最安値業者と契約

**リスクと機会**
- **リスク**: 準備期間がほとんどない。仕様書の詳細確認が困難
- **機会**: 競合が少ない。大手が参入しない規模感で地元企業にチャンス

AIは公告日から締切までの日数が極端に短い（3日以内）案件を発見した場合、「緊急性の高いオープンカウンタまたは随意契約の可能性」として警告を出すべきです。

### AIによる判別ロジック

検索結果の`procedure_type`や案件名のキーワードから、以下のように入札方式を推定できます:

```
IF "WTO" IN 案件名 THEN
  → 一般競争入札（大型・長期戦）
  → 公告期間40日以上を確認
  → 外資系企業の参入可能性を示唆

ELSE IF "指名" OR "公募型" IN 案件名 THEN
  → 指名競争入札
  → 名簿登載の確認を促す
  → 技術資料の準備期間を考慮

ELSE IF (公告日 - 締切日) <= 3 days THEN
  → 随意契約・オープンカウンタ
  → 緊急対応を警告
  → 過去の見積実績があれば有利

ELSE
  → 一般競争入札（標準）
  → 参加資格確認申請の締切（通常10〜15日後）を最優先で確認
```

## AIアドバイス例

**ケース1: WTO案件の発見**
```
「この案件はWTO対象の大型案件です。公告期間が40日確保されていますが、
Aランク（P点1000点目安）かつ特定建設業許可が必須です。国際競争となるため、
英文での質問対応や、外資系企業との価格競争も想定されます。」
```

**ケース2: 指名型案件**
```
「この案件は指名型です。事前の名簿登録がない場合、参加は困難です。
○○市の入札参加資格名簿への登載状況を確認してください。
次回の名簿更新は来年4月です。」
```

**ケース3: 緊急オープンカウンタ**
```
「公告から締切まで48時間しかありません。これは緊急性の高いオープンカウンタ
と思われます。仕様書の確認と見積作成を最優先で行ってください。」
```
