# 法的制約と最低制限価格（Legal Constraints and Floor Price）

## Brief

### 最低制限価格とは

**ダンピング（不当廉売）を防ぐための「最低ライン」**

- この価格を下回ると、どれだけ安くても即失格
- 予定価格の約70〜90%に設定されることが多い
- 自治体ごとに異なる計算式（変動型が主流）

### 重要ポイント

- **計算式の確認**: 自治体の「最低制限価格算定要領」を事前入手
- **安易な値引きは危険**: 失格リスクを常に念頭に
- **低入札調査**: 最低制限価格付近だと詳細審査の対象

## Detailed

### 2.4 「落札の罠」：最低制限価格と低入札調査

**概要**

公共入札には、ダンピング（不当廉売）を防ぐための**最低制限価格（Floor Price）**が存在します。この価格を下回ると、どれだけ安くても即座に失格となります。

**計算式（一般的モデル）**

$$\text{最低制限価格} \approx (\text{直接工事費} \times 0.97 + \text{共通仮設費} \times 0.90 + \text{現場管理費} \times 0.90 + \text{一般管理費} \times 0.55)$$

※自治体により係数は異なります（通称：**変動型最低制限価格**）

**変動型とは**

予定価格に対して一律70%などと固定せず、工事の内訳（直接工事費、諸経費等）の比率に応じて算定する方式。これにより「談合」や「予測入札」を防止しています。

**AIのアドバイス**

AIはこの計算式を提示し、以下のように警告する必要があります：

```
「安易な値引きは失格リスクを高めます。発注自治体が公表している
『積算基準』と『最低制限価格算定要領』を確認し、係数をシミュレーション
してください。」
```

特に、計算結果がランダム係数で変動する自治体（くじ引き要素がある場合）はその旨も伝えます。

### 低入札調査制度

最低制限価格を設けない代わりに、極端に安い価格（調査基準価格を下回る）を提示した業者に対して、「その価格で適正に履行できるか」を詳細審査する制度です。

**調査基準価格の計算式**

$$\text{調査基準価格} \approx \text{直接工事費} \times 0.95 + \text{共通仮設費} \times 0.90 + \text{現場管理費} \times 0.80 + \text{一般管理費} \times 0.30$$

（最低制限価格よりやや高めに設定されることが多い）

**審査内容**
1. 原価計算書の提出と説明
2. 配置予定技術者の確保証明
3. 下請業者との契約内容
4. 資材調達の見積書

**リスク**
- 審査に2〜4週間かかる
- 不適格と判断されれば失格
- 審査中は他の入札に参加できない（技術者の二重配置不可）

**AIのアドバイス**

```
「あなたの入札価格は予定価格の78%で、調査基準価格（約80%）を下回る
可能性があります。低入札調査の対象となった場合、詳細な原価計算書と
下請見積の提出が求められます。準備に自信がない場合は、価格を82%程度に
引き上げることを推奨します。」
```

## Comprehensive

### 最低制限価格制度の完全ガイド

#### 制度の背景と法的根拠

**導入の経緯**

1990年代、公共工事の過度な価格競争により、品質の低下や下請業者へのしわ寄せ（買い叩き）が問題となりました。これを是正するため、地方自治法施行令が改正され、最低制限価格制度が導入されました。

**法的根拠**
- 地方自治法施行令第167条の10
- 国土交通省「工事請負契約における最低制限価格の算定方法について」（通知）

#### 最低制限価格の詳細計算式

**基本構造**

予定価格の内訳は以下の要素で構成されます：

```
予定価格 = 直接工事費 + 共通仮設費 + 現場管理費 + 一般管理費 + 消費税
```

最低制限価格は、これらの要素に個別の係数を掛けて算出します。

**標準的な係数（国土交通省モデル）**

| 費目 | 係数 | 理由 |
| --- | --- | --- |
| 直接工事費 | 0.95〜0.97 | 材料費・労務費は削減余地が少ない |
| 共通仮設費 | 0.90 | 一定の削減可能性あり |
| 現場管理費 | 0.80〜0.90 | 効率化で削減可能 |
| 一般管理費 | 0.30〜0.55 | 企業努力で大幅削減可能 |

**計算例**

予定価格5,000万円の道路改良工事：

| 費目 | 金額 | 係数 | 算定額 |
| --- | --- | --- | --- |
| 直接工事費 | 3,000万円 | 0.97 | 2,910万円 |
| 共通仮設費 | 500万円 | 0.90 | 450万円 |
| 現場管理費 | 800万円 | 0.90 | 720万円 |
| 一般管理費 | 700万円 | 0.55 | 385万円 |
| **合計** | **5,000万円** | - | **4,465万円** |

→ 最低制限価格 = 4,465万円（予定価格の約89.3%）

**4,464万円で入札 → 失格**
**4,466万円で入札 → 有効**

#### 変動型最低制限価格の戦略的意味

**固定型の問題点**

従来の「予定価格の70%」などの固定型では：
- 入札業者が最低制限価格を容易に予測できる
- 全員が同じ価格（最低制限価格ギリギリ）で入札 → くじ引きで決定
- 談合の温床となる

**変動型のメリット**

工事の内訳構成によって最低制限価格が変動するため：
- 予測が困難 → 真剣な積算が必要
- 各社が独自の価格を提示 → 真の競争が実現
- 談合防止効果

#### 自治体別の係数バリエーション

| 自治体 | 直接工事費 | 共通仮設 | 現場管理 | 一般管理 | 結果（予定価格比） |
| --- | --- | --- | --- | --- | --- |
| 国土交通省 | 0.97 | 0.90 | 0.90 | 0.55 | 約87〜92% |
| 東京都 | 0.95 | 0.90 | 0.85 | 0.50 | 約85〜90% |
| 地方自治体A | 0.97 | 0.90 | 0.90 | 0.60 | 約90〜93% |
| 地方自治体B | 0.95 | 0.85 | 0.80 | 0.40 | 約82〜88% |

**ランダム係数の導入**

一部の自治体では、さらに予測を困難にするため、係数に±5%のランダム変動を加えています。

例：現場管理費の係数が0.85〜0.95の範囲でランダムに決定される

**AIの判別ロジック**

```python
def estimate_floor_price_range(notice_detail: dict) -> tuple:
    # 自治体の係数テーブルを取得（データベース化が理想）
    coefficients = get_coefficient_by_location(notice_detail["location"])

    # 予定価格の内訳を推定（公表されていない場合は標準比率を使用）
    breakdown = estimate_cost_breakdown(notice_detail["estimated_price"])

    # 最低制限価格の計算
    floor_min = (
        breakdown["direct"] * coefficients["direct_min"] +
        breakdown["common"] * coefficients["common_min"] +
        breakdown["site_mgmt"] * coefficients["site_min"] +
        breakdown["general_mgmt"] * coefficients["general_min"]
    )

    floor_max = (
        breakdown["direct"] * coefficients["direct_max"] +
        breakdown["common"] * coefficients["common_max"] +
        breakdown["site_mgmt"] * coefficients["site_max"] +
        breakdown["general_mgmt"] * coefficients["general_max"]
    )

    return (floor_min, floor_max)
```

**出力例**:
```
「この案件の最低制限価格は、4,350万円〜4,600万円の範囲と推定されます。
入札価格は4,700万円以上を推奨します。」
```

#### 低入札調査制度の実務

**調査基準価格の設定**

最低制限価格よりやや高めに設定され、これを下回ると「低入札調査」の対象となります。

**調査プロセス**

1. **書類提出（落札候補者決定後3日以内）**:
   - 詳細な原価計算書
   - 配置予定技術者の雇用証明
   - 下請業者との見積書・契約書（案）
   - 資材調達先の見積書

2. **ヒアリング（1〜2週間後）**:
   - 発注者の審査会で説明
   - 「なぜこの価格で可能か」を論理的に説明

3. **審査結果（2〜4週間後）**:
   - 適格 → 契約
   - 不適格 → 失格（次点者が繰り上がり）

**不適格となる典型例**

- 下請業者の見積が極端に安く、実現可能性が疑われる
- 配置予定技術者が他の現場と重複している
- 材料費の見積が市場価格より著しく低い
- 労務費が最低賃金を下回る

**AIのリスク警告**

```
「低入札調査の対象となる可能性が高いです。以下を確認してください：
1. 下請業者からの正式な見積書（口約束ではNG）
2. 配置技術者が他の現場と重複していないか
3. 労務費が最低賃金＋社会保険料を上回っているか

審査に不合格となると、次回以降の入札で不利になる可能性があります。」
```

#### くじ引き（抽選）のメカニズム

**発生条件**

複数の業者が全く同じ価格で入札した場合、くじ引き（電子抽選）で落札者を決定します。

**変動型導入後の状況**

以前は「最低制限価格ピッタリ狙い」で多数のくじ引きが発生していましたが、変動型導入後は大幅に減少しました。ただし、小規模案件（500万円以下など）では依然としてくじ引きが発生します。

**AIのアドバイス**

```
「この案件は少額（300万円）のため、最低制限価格が固定型（70%）と
推定されます。210万円ピッタリで入札する業者が複数いる可能性が高く、
くじ引きとなる確率が50%以上です。確実な受注を望む場合は、やや高めの
215万円での入札も検討してください。」
```
