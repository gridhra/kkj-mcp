# タイムラインと法的期間（Timelines and Legal Periods）

## Brief

### 重要な締切

- **公告期間**: 原則10日以上（地方自治法施行令167条）
- **申請締切**: 公告から約10〜15日後（最重要！）
- **見積期間**:
  - 500万円未満: 1日以上
  - 500万円〜5,000万円: 10日以上
  - 5,000万円以上: 15日以上（建設業法20条）

### AIの警告タイミング

- 公告から12日経過 → 「申請締切が迫っています（48時間以内の可能性）」
- 入札日まで7日以内 → 「短縮特例が適用されている可能性」
- 年度末（1-3月） → 「3月末工期の可能性、資材調達確認を」

## Detailed

### 2.2 時間軸の解剖学：公告から契約までの法的タイムライン

AIが提供できる最大の付加価値は、日付データの羅列を「意味のあるスケジュール」に変換することです。これには地方自治法施行令等の法的根拠への深い理解が不可欠です。

### 2.2.1 標準的なフロー（地方自治体・一般競争入札）

例えば、予定価格3,000万円の道路改良工事を想定します。地方自治法施行令第167条の6に基づき、以下のタイムラインが形成されます。

| フェーズ | 期間（目安） | 内容と法的根拠 |
| --- | --- | --- |
| 1. 公告 | Day 0 | KKJ API等で情報公開 |
| 2. 質問期間 | Day 1〜10 | 設計図書（仕様書）の閲覧と質疑応答 |
| 3. 資格確認申請 | Day 14前後 | **【重要】** 参加表明のデッドライン。ここを逃すと失格 |
| 4. 入札（投函） | Day 20〜25 | 見積金額の提出。電子入札または郵送 |
| 5. 開札 | Day 25 | 金額の公開と落札候補者の決定 |
| 6. 事後審査 | Day 26〜30 | 落札候補者の資格要件の詳細審査 |
| 7. 契約締結 | Day 35 | 正式契約 |

**AIの推論ロジック**

検索された案件の`cft_issue_date`（公示日）が現在日から12日経過している場合、AIは単に案件を表示するだけでなく、以下のように補足すべきです：

```
「警告：標準的な資格確認申請期限（公告から約2週間）が迫っています。
地方自治法施行令に基づく一般的なスケジュールでは、あと48時間以内に
申請が必要な可能性があります。」
```

### 2.2.2 期間の短縮特例（Emergency / Shortening）

**法的根拠**: 地方自治法施行令第167条の6第2項

法律は原則として10日以上の公告期間を求めていますが、「急を要する場合」は5日まで短縮することができます。

**シグナル**:
- 災害復旧工事
- 年度末（3月）の予算消化案件

**AIの洞察**

公告日から入札日までの期間が極端に短い（1週間以内）案件を発見した場合、AIはそれを「緊急発注案件」と識別し、以下のようにアドバイスします：

```
「この案件は特例措置により期間が短縮されています。質問期間が設けられて
いないか、極めて短い可能性があります。仕様書の詳細確認を最優先で
行ってください。」
```

### 2.2.3 建設業法による「見積期間」の保護

**法的根拠**: 建設業法第20条第3項

発注者（または元請）は、受注者（下請）に対して適切な見積期間を与える法的義務があります。

| 案件規模 | 原則期間 | やむを得ない場合 |
| --- | --- | --- |
| 500万円未満 | 1日以上 | - |
| 500万円〜5,000万円 | 10日以上 | 5日 |
| 5,000万円以上 | 15日以上 | 10日 |

**AIの戦略的アドバイス**

ユーザーが元請として入札に参加する場合、AIは次のように助言します：

```
「この案件は5,000万円規模が見込まれます。建設業法上、下請業者に見積を
依頼する際は最低10日間の猶予を与える必要があります。入札締切から逆算して、
直ちに協力会社への打診を開始してください。」
```

## Comprehensive

### 法的タイムラインの完全ガイド

#### 地方自治法施行令による期間規制

**第167条の6（一般競争入札）**

```
第一項：一般競争入札により契約を締結しようとするときは、
         政令の定めるところにより、その入札期日の前日から起算して
         少なくとも十日前に公告しなければならない。

第二項：前項の期間は、急を要する場合においては、五日まで
         短縮することができる。
```

**第167条の6の2（電子入札の特例）**

電子入札システムを利用する場合、公告期間は以下のように柔軟化されます：
- システムが常時アクセス可能な場合、公告から入札まで5日でも適法
- ただし、実務上は10日以上が一般的

#### WTO案件の特別ルール

**政府調達協定（GPA）による規制**

WTO対象案件（工事6.8億円以上等）では、国内法よりも厳格な期間規制が適用されます：

1. **公告期間**: 40日以上（官報掲載が必須）
2. **質問期間**: 入札日の10日前まで質問可能
3. **英文対応**: 外国企業からの質問に英語で回答する義務

**AIの判別ロジック**

```
IF "WTO" IN 案件名 OR 予定価格 >= 680,000,000円 THEN
  → 公告期間が40日未満の場合、データ不備を疑う
  → 国際競争を想定し、競合分析の重要性を強調
```

#### 建設業法による見積期間の詳細

**第20条第3項の完全条文**

```
注文者は、請負契約を締結する場合において、工事内容、工期等について、
受注者に対し、適切な見積期間を与えなければならない。
```

**国土交通省告示による具体的期間**

| 請負代金の額 | 見積期間（原則） | 特例 |
| --- | --- | --- |
| 500万円未満 | 1日以上 | - |
| 500万円以上 5,000万円未満 | 10日以上 | やむを得ない場合5日 |
| 5,000万円以上 | 15日以上 | やむを得ない場合10日 |

**実務上の影響**

元請として入札に参加する場合、この見積期間を確保できないと：
1. 下請業者の選定が不十分になり、品質リスクが高まる
2. 建設業法違反として行政処分の対象となる可能性
3. 下請業者から不当な価格での受注を強いられたとして訴えられるリスク

**AIの高度なアドバイス例**

```
「この案件の入札日は10日後です。予定価格が5,000万円を超えると想定されるため、
建設業法上、下請業者への見積依頼は本日中に行う必要があります。

推奨アクション：
1. 今日中に主要協力会社3社以上に見積依頼
2. 5日後（法定最短期間）に見積回収
3. 残り5日で積算と技術提案書作成

なお、10日間の見積期間を確保できない場合、下請契約後に建設業法違反を
問われるリスクがあります。」
```

#### 年度区分と工期の関係

**会計年度独立の原則**

日本の公共事業予算は4月〜翌年3月の「年度」単位で管理されます。このため：

- **年度内完成が原則**: 3月31日までに完成・検査・支払いが必要
- **繰越明許費**: 年度内に完成しない場合、議会の議決で翌年度に繰り越し可能
- **債務負担行為**: 複数年度にわたる契約（大型プロジェクト）に適用

**AIによる季節性分析**

```python
def analyze_timeline_risk(issue_date: datetime, bid_date: datetime, current_date: datetime) -> str:
    days_since_issue = (current_date - issue_date).days
    days_until_bid = (bid_date - current_date).days

    # 申請締切リスク
    if 10 <= days_since_issue <= 14:
        return "【緊急】資格確認申請の締切が2日以内に迫っている可能性が高いです"

    # 短縮特例の疑い
    if days_since_issue + days_until_bid <= 7:
        return "【注意】公告期間が極端に短く、緊急案件の可能性があります"

    # 年度末リスク（1-3月）
    if current_date.month in [1, 2, 3]:
        return "【年度末】工期が3月末固定の可能性。資材調達と技術者手配を最優先で確認"

    return "標準的なスケジュールです"
```

## AIアドバイス例

**ケース1: 申請締切が迫っている**

```
「公告日が12月1日、現在12月13日です。標準的な一般競争入札では、
資格確認申請の締切は公告から約2週間（12月14日頃）に設定されることが
多いです。発注機関のウェブサイトで申請締切を至急確認してください。」
```

**ケース2: 短縮特例の疑い**
```
「公告から入札まで5日間しかありません。これは地方自治法施行令第167条の6第2項
『急を要する場合』の短縮特例が適用されていると思われます。質問期間が
設けられていない可能性が高いため、仕様書の不明点は電話で直接確認する
ことをお勧めします。」
```

**ケース3: 建設業法の見積期間**
```
「この案件は予定価格8,000万円、入札日が10日後です。元請として参加する場合、
建設業法により下請業者に最低10日間の見積期間を与える義務があります。
本日中に協力会社へ見積依頼を開始しないと、法令遵守が困難になります。」
```

**ケース4: 年度末の工期リスク**
```
「現在2月中旬で、この案件の工期が『契約日から60日』と記載されています。
契約が3月上旬と仮定すると、実質的な作業期間は3月末までの3週間程度に
圧縮される可能性があります。年度内完成が求められる場合、資材の即納可否と
技術者の即時配置が入札判断の鍵となります。」
```
